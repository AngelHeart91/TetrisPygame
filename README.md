# Tetris Pygame

Learning Python and pygame by building a Tetris game.

## Update/Features
1 - Create the game scene

2 - Create the Tetromino Shape with a grid

3 - Move it on the grid (key : a, d, left, right)

4 - Save the tetromino it in the grid

5 - Down the tetromino at the bottom directly (key : space bar)

## How to run
```bash
python main.py
